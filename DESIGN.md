# TODO App — Design System
## Visual Style Guide

**Version:** 1.0  
**Last Updated:** March 2026  
**Style Reference:** Obsidian × Spotify

---

## 1. Design Philosophy

### Core Principles
- **Minimal First:** Every element must justify its existence. Remove decorative noise.
- **Content is King:** Tasks and dates are the heroes. UI recedes into the background.
- **Dark by Default:** Optimized for long focus sessions. Light mode is secondary.
- **Smooth & Subtle:** Animations should feel natural, never distracting.
- **Keyboard Friendly:** Power users should navigate without touching the mouse.

### Inspiration
| Reference | What We Borrow |
|-----------|----------------|
| **Obsidian** | Clean typography, markdown aesthetic, vault-like organization, minimal chrome |
| **Spotify** | Dark depth, accent color usage, card-based layouts, smooth hover states |
